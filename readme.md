# BigWheels

A cool game where you rollerblade down a hill and jump and do stunts together.

![Whoa!](/bigwheels.png?raw=true)

## Build

Install Java and [maven](https://maven.apache.org/)

```bash
./build.sh
```

## Run

```bash
./run.sh
```
